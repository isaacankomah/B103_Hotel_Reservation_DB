# Video Demonstration Plan

Target length: 3 to 5 minutes.

## Suggested Flow

1. Introduce the project and domain.
2. Show the ER diagram and explain the relationships.
3. Open the SQL schema file and explain the tables.
4. Run or show sample data insertion.
5. Demonstrate key queries:
   - list bookings with customer and room details
   - find available rooms
   - calculate total revenue
   - show booking counts by room type
6. Conclude with what the database achieves and one future improvement.
